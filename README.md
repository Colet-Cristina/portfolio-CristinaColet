# ⛩️ Portfolio Dojo Digital — Cristina Colet

¡Bienvenida/o! Soy Fullstack Developer Junior (Adalab) con una trayectoria que mezcla la paciencia de la naturaleza y la agilidad de la gestión en entornos reales.

Este proyecto es una Single Page Application (SPA) moderna y responsiva, diseñada para mostrar mi trayectoria, proyectos y habilidades.

Está inspirada en la estética tradicional japonesa, reflejando disciplina, equilibrio y una profunda dedicación al arte de programar.

> "Cultivando soluciones, gestionando entornos, escribiendo código."

---

## 🌸 La Filosofía del Proyecto (Las Semillas de Código)

Este portfolio es la narrativa de mi transición hacia el desarrollo tech, basada en tres pilares:

- **Paciencia Forestal:** Como viverista, aprendí que las raíces (el código base) deben ser sólidas para que el proyecto crezca sano y robusto.
- **Gestión bajo presión:** Mi etapa en comedores escolares me dio la agilidad y organización necesarias para resolver imprevistos, tomar decisiones rápidas y trabajo en equipo.
- **Artesanía Digital:** Un enfoque en el detalle, la limpieza del código y la armonía visual de la maquetación.

---

## 🛠️ Arquitectura del Proyecto

Para materializar esta filosofía, el portfolio sigue una estructura limpia, modular y escalable de Single Page Application (SPA), basada fielmente en las buenas prácticas aprendidas en **Adalab**:

```text
├── components/          # Componentes visuales reutilizables
│   ├── layout/          # Estructuras globales (Header, Footer)
│   └── GardenFlower.jsx # Componentes interactivos del jardín
│
├── pages/               # Secciones principales de la SPA (Vistas)
│   ├── Hero.jsx         # Bienvenida y presentación potente
│   ├── Roots.jsx        # Mis raíces (Bio e Historia)
│   ├── GardenCode.jsx   # El jardín de código (Showroom de proyectos)
│   └── Tools.jsx        # Tecnologías y herramientas (El Dojo de habilidades)
│
├── services/            # Capa de datos y lógica dinámica
│   ├── projectsData.json
│   └── skillsData.json
│
└── styles/              # Arquitectura SASS 7-1 (100% rem y BEM)
    ├── core/            # Variables globales, reset y animaciones
    ├── layout/          # Estilos de la estructura general
    └── pages/           # Estilos específicos de cada sección del Dojo

```
